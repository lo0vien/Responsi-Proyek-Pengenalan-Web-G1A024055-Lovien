##  Deskripsi Website

Website ini  berbasis HTML, CSS, dan JavaScript yang menggabungkan dua fitur utama:

### 1. Digital Persona Finder

Fitur interaktif yang memungkinkan pengguna:

* Menginput nama
* Mengisi hobi
* Memasukkan angka keberuntungan (1–10)

Kemudian sistem akan menampilkan persona digital berdasarkan input tersebut.

---

### 2. Profile (Linktree Style)

Halaman profil pribadi yang berisi:

* Foto profil
* Biodata singkat
* Link ke media sosial dan platform lainnya:

  * Instagram
  * WhatsApp
  * GitHub
  * Personal Website
